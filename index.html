About the Empire

Built by musicians, run like a label should be.

KVNGS EMPIRE started as a single booth and a stubborn belief that great records shouldn't require a major-label budget. Today we're a full recording studio and independent record label: engineers, producers, A&R and a roster of artists working under one roof toward the same standard — records that hold up anywhere they're played.

Every session is treated like a release. Every mix is treated like it's going to radio. That's the only tier of work we do.
