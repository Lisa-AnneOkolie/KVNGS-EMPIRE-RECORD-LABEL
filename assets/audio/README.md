# Audio files go here

The audio player on the site looks for these files:

- track-1.mp3
- track-2.mp3
- track-3.mp3
- track-4.mp3

Drop your real tracks in this folder using those exact filenames, or open
`js/script.js` and edit the `TRACKS` array (title, artist, length, and the
`src` path) to point at whatever filenames/paths you use.
